# Pygame dla Ciebie

Witaj w repozytorium poświęconym tworzeniu gier w języku **Python** przy użyciu biblioteki **Pygame**. Znajdziesz tutaj pliki źródłowe, zasoby oraz kody powstałe na podstawie lekcji z kanału **Samuraj Programowania**.

---

## O projektach

Głównym celem tego repozytorium jest dobra zabawa, a przy okazji nauka podstaw tworzenia gier i praktyczne wykorzystanie języka Python.

## Jak pobrać i uruchomić?

### 1. Pobieranie plików

Możesz pobrać to repozytorium na dwa sposoby:

- **Bezpośrednie pobranie:**
  Kliknij zielony przycisk "Code" na górze strony i wybierz "Download ZIP". Następnie wypakuj archiwum na swoim komputerze.

- **Przez Git (zawansowane):**
  Otwórz terminal i wpisz:

  ```bash
  git clone https://github.com/bartibo/pygame-for-you.git
  ```

### 2. Instalacja wymagań

Zanim uruchomisz gry, upewnij się, że masz zainstalowanego Pythona oraz bibliotekę Pygame. Możesz ją zainstalować komendą:

```bash
pip install pygame
```

### 3. Uruchamianie gry

Wejdź do folderu z konkretnym projektem i uruchom główny plik (zazwyczaj `main.py`):

```bash
python main.py
```
